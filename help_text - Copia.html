Introdução

Este constutor foi criado para facilitar a criação de informes a partir de dados padronizados. Sua operação permite que modelos de ralaório definidos préviamente sejam criados em poucos minutos a partir de bases de dados (em Excel) estruturados de modo compatível. Seguem algumas informações iniciais para que o usuário possa começar a utilizá-lo.

ESTE SOFTWARE ESTÁ EM FASE BETA. Sua estabilidade ainda é baixa e mais funcionalidades serão acrescentadas so longo do tempo.

Esta tela - "Preview" - apresentará uma versão amostral do informe, uma vez que sejam configuradas as informações de formatação e de aquisição dos dados. Caso presencie algum bug nela, clique em "Recarregar" para recarregar a visualização.

Primeiros Passos

Informes são construídos a partir de cards: blocos de informação contendo uma campo de 'Cabeçalho', uma imagem inicial de 'Banner',um 'Título', um 'Corpo', um 'Rodapé', que pode ser uma imagem ou um texto, e uma borda que engloba o conteúdo. Toda imagem também possui um campo de legenda atribuído e opções para inserção de hyperlinks.

Adcionando um Card

Cards podem ser criados individualmente ou automaticamente a partir de uma base de dados. Em ambos os casos, os campos a serem preenchidos são semelhantes, porém a utilização do método de importação permite que bases já existentes sejam aproveitadas sem a nescessidade de transcrição manual.

Criação individual ou manual

Navegando nas opções Adicionar Cards>Adicionar Manualmente, o programa exibirá a tela de adição de cards.

Nesta tela, pode-se configurar os conteúdos individuais de cada campo dispnível. Use as etiqutas no topo para navegar entre os grupos de configurações.

Nas seções Banner e Rodapé, imagens podem ser importadas da internet ou carregadas do armazenamento local.

  • Imagens Web: selecione a opção URL e cole o link no campo de texto; Imagens adicionaadas deste modo não ocuparão memória no e-mail mas são dependentes do link estar em funcionamento no momento da leitura da mensagem.
  • Imagens Locais: selecione "Caminho" e clique na opção "Buscar Imagem" para selecionar um arguivo da memória. Formatos suportados são .png, .jpg/.jpeg e .gif. Imagens adicionadas deste modo são anexadas diretamente ao email após convertidas em formato base64 (via CID) e, portanto, ocupam espaço de memória no e-mail. Não são todavia dependentes de disponibilidade da original como o ocorre nas imagens Web. Imagens muito gradnes e complexas demorarão para serem carregadas pelo usuário. Alguns provedores de e-mail podem também destacá-las como anexos ao fim da mensagem.

Os campos Legenda e Hyperlink adicionam, respectivamente, um texto de legenda abaixo da imagem e um link qualquer a ser acessado quando se clica na imagem adicionada.

Para a seção de Rodapé, pode-se também, no lugar de uma imagem, adicionar-se um texto.

Uma vez configuradas todas as opções, clique no botão Adicionar para gerar um card com estas configurações e adicioná-lo ao fim do documento, ou Inserir para criá-lo em uma posição específica do documento (por exemplo, o segundo card em uma lista de três). Cards posicionados no local onde um novo será inserido são reposicionados uma posição a frente (por exemplo: em uma lista de três cards, ao posicionar um novo na primeira posição, o primeiro carda tornar-se-á o segundo, o terceiro se tornará o quarto e assim por diante).

Configurações de formatação podem também ser realizadas a partir da opção Formatação Avançada. Detalhes acerca de como realizar esta sorte de customização são encontrados na seção Formatação e Exibição.

Criação por Importação de Base dados

Navegando nas opções Adicionar Cards>Importar de Planilha, o programa exibirá a tela de importação de cards.

Nesta tela, pode-se configurar os parâmetros de importação dos dados. Utilize as abas da parte superior para navegar por entre os grupos de configurações disponíveis.

Estruturando uma planilha de dados

Criar uma planilha para utilização com este programa é uma tarefa simples. Para cada conjunto de dados que comporá um card, deve-se inseri-las em uma mesma linha (isto é, cada linha representa um card), de modo que as colunas contemplem cada um dos seguintes campos:

  • Dados Obrigatórios
    • Corpo
  • Dados Opcionais
    • Cabeçalho
    • Título
    • Conteúdo do Banner
    • Link do Banner
    • Legenda do Banner
    • Conteúdo do Rodapé
    • Link do Rodapé
    • Legenda do Rodapé
    • Datas
    • Indicador de Inclusão

Colunas de dados opcionais só são nescessárias quando sua funcionalidade for desejada. Além disso, colunas não precisame estar em ordem ,porém em ordem conhecida, e podem possuir qualquer nome sem restrições. Qualquer número de linhas de cabeçalho podem ser utilizadas, mas uma vez que as linhas de dados comecem, é desaconselhável que se utlize as linhas posteriores para outros fins que não o armazenamento de mais dados. Colunas que não as reservadas para os dados podem ser utilizadas livremente. A página da pasta de trabalho do Excel que contém os dados pode possuir qualquer nome e estar em qualquer poisção, mas essas informações devém ser conhecidas.

Planilha

Na seção Caminho da Planilha, clique no botão Buscar para buscar a planilha em Excel contendo os dados estruturados; Ou copie diretamente o caminho do arquivo (incluindo seu nome e extensão .xlsx). Em Nome ou Número da Aba insira o nome ou a posição (iniciada em 0) da aba da pasta de trabalho contendo os dados estruturados. Por fim, em Linhas de Cabeçalho quantas linhas estão sendo ocupadas pelo cabeçalho (e pelas linhas anteriores a ele, se aplicável) para que o progra as desconsidere durante a importação.

Colunas

Para cada coluna implementada na planilha de dados, entre a letra correspondente no campo indicado. Caso deseje que todos os cards possuam o mesmo conteúdo em um dado campo, entre este dado nas opções "padrão". Por padrão, se ambos os campos forem preenchidos, o valor padrão tem prevalência sobre os dados importados e os subtistiuirá em todos os cards. Deixar ambos os campos em branco remove o espaço dedicado a este componente no card.

Figuras

Nesta aba deve-se informar ao programa como deve-se tratar as imagens adicionadas aos campos Banner e Rodapé.

  • Seleção de tipo: Define o método adotado para importar imagens para o e-mail. Escolher entre URL (link de imagem web), Caminho (Caminho de imagem local) ou Texto (somente para rodapé, tratar dados importados como texto, não imagem;
  • Coluna do link: Letra da coluna na planilha que contém os links para os endereços web para os quais o usuário será encaminhado ao clicar sobre a mensagem. Opção padrão: será aplicada em todos os cards e substuirá os dados importados;
  • Coluna da Legenda: Letra da coluna na planilha que contém o texto de elgenda a ser inserido embaixo de cada imagem. Opção padrão: será aplicada em todos os cards e substuirá os dados importados; Opção "Texto se vazio", substitui todo espaço em branco nos dados importados pelo texto inserido.

Tags e HTML

O sintetizador do texto utiliza linguagem HTML, de modo que todas as tags HTML compatíveis com o provedor de e-mail adotado irão funcionar. Adiconalmente, para tornar a escrita mais prática, escrever "<T>" antes de um parágrafo fará com que todo o parágrafo seja formatado como um "Subtítulo". Semelhante modo, parágrafos iniciados com uma tabulação, escrevendo "<tab>", são interpretados como uma lista de items

Formatação e Exibição

Devido à limitações técnicas, não é possível exibir bordas arredondadas nesta tela de preview, porém estas estarão presentes em provedores de e-mail. Como exceção está O OUTLOOK PARA DESKTOP que não possui suporte para a maioria das funções HTML. Demais provedores devem operar normalmente.

Opções de formatação padronizadas estão disponíveis nas páginas de importação e criação manual de cards.
